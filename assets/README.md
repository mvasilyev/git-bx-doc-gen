# Icon Placeholders

This directory should contain the extension icons:

- `icon16.png` - 16x16 pixels (toolbar icon)
- `icon32.png` - 32x32 pixels (extension management)
- `icon48.png` - 48x48 pixels (extension management)
- `icon128.png` - 128x128 pixels (Chrome Web Store)

For now, you can use any PNG icons or create them using online tools.

The icons should represent documentation/book themes to match the extension's purpose.

Example icon sources:
- Flaticon.com
- Icons8.com
- Google Material Icons
- Custom design tools

Remember to ensure you have the rights to use any icons in your extension.
